## Examples

### Using inset prop

The `inset` prop provides a way to add left padding to menu items. Here's how it works:

<CodeViewer
  className="mt-3"
  code={`// Adding inset to MenuPopup affects all items
<MenuPopup inset> // [!code highlight] 
    <MenuItem>All items</MenuItem>
    <MenuItem>Have padding</MenuItem>
</MenuPopup>

// Adding inset to specific MenuItems
<MenuPopup>
    <MenuItem>Normal item</MenuItem>
    <MenuItem inset>Only this has padding</MenuItem> // [!code highlight] 
    <MenuItem>Normal item</MenuItem>
</MenuPopup>`}
/>
